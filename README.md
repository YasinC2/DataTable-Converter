# DataTable Converter

A powerful, client-side data format converter tool that runs entirely in your browser. Convert between JSON, CSV, XML, HTML tables, and Markdown tables with ease.

## ✨ Features

- **Multiple Format Support**: Convert between JSON, CSV, XML, HTML Tables, and Markdown Tables
- **Smart Auto-Detect**: Automatically detects the input format from your data
- **Real-time Conversion**: Results update instantly as you type
- **File Upload**: Drag & drop or browse to upload single or multiple files
- **Batch Processing**: Convert multiple files at once and download as ZIP
- **Tree View**: Visualize JSON data in an interactive tree structure
- **Live Statistics**: See file size, line count, and word count in real-time
- **Syntax Highlighting**: Beautiful code highlighting with copy support
- **Dark/Light Theme**: Toggle between themes with persistent storage
- **Advanced CSV Options**: Customize delimiters (comma, semicolon, tab, pipe)
- **Sample Data**: One-click samples to get started quickly
- **Fully Client-Side**: No server required, your data never leaves your browser

## 🚀 How to Use

1. **Select Input Format** - Choose from dropdown or use "Auto Detect"
2. **Enter or Upload Data** - Type directly or drag & drop a file
3. **Choose Output Format** - Select your desired format
4. **Get Results** - Conversion happens instantly in the output panel
5. **Download or Copy** - Save as file or copy to clipboard

## 🛠️ Built With

- Vanilla JavaScript (ES6+)
- [PapaParse](https://www.papaparse.com/) - Powerful CSV parsing
- [xml-js](https://github.com/nashwaan/xml-js) - XML to JSON conversion
- [Highlight.js](https://highlightjs.org/) - Syntax highlighting
- [JSZip](https://stuk.github.io/jszip/) - ZIP file generation
- [Font Awesome](https://fontawesome.com/) - Icons

## 📦 Installation

Simply clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/YasinC2/DataTable-Converter.git
cd DataTable-Converter
# Open index.html in your browser
```

No build process or dependencies to install!

## 💻 Local Development

1. Clone the repository
2. Make changes to `index.html`, `main.css`, or `main.js`
3. Refresh the browser to see changes
4. All libraries are loaded via CDN, no local setup needed
